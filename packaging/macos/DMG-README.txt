iTerm CD for iTerm2

推荐安装方式:
1. 双击 iTerm CD Installer.pkg
2. 安装完成后重启 iTerm2
3. 在 iTerm2:
   Settings > Profiles > Session > 打开 Status bar enabled
   Configure Status Bar > 拖入 CD To Current Directory

其他文件:
- iTerm CD Installer.app: 免 pkg 的双击安装器
- iTerm CD Uninstaller.app: 双击卸载器

说明:
- 需要启用 iTerm2 Shell Integration 才能准确获取当前目录
- 卸载后顶部 badge 会尽量立即清理，状态栏组件和标题提供器需要重启 iTerm2 后消失
